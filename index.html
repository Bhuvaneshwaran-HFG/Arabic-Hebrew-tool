Why reverse Hebrew for Unity? — Unity's TMP doesn't natively handle RTL BiDi. Hebrew input typed left-to-right stores as logical order. To display correctly in Unity TMP (which renders left-to-right), you reverse the word order of each line. Individual letter order stays intact — Hebrew glyphs don't need reshaping like Arabic does.
